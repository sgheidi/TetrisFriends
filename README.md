# Fruit Tetris

A simple, elegant Tetris implementation written in OpenGL/C++.

## Setting up

Playing the game is as easy as running ``` make && ./FruitTetris ```!

## How to play

- 'p': pause

- 'q' or 'ESC': exit

- 'r': restart

- Up key: rotate piece

- Left/right keys: horizontal movement

- Down key: downward acceleration
